[−][src]Module telamon::explorer::mcts

Search space exploration using MCTS-style algorithm

This module provides functionality for exposing branch-and-bound candidate trees, as well as MCTS-style algorithms for exploring them.

The trees are defined using Node and Edge, and a trace-enabled NodeCursor is provided to move in the tree while recording a sequence of events.

The MctsWalker, parameterized with the appropriate bandit and rollout policies, can be used to explore the resulting search tree in an MCTS-like fashion.
