pub enum NewNodeOrder {
    Api,
    Random,
    Bound,
    WeightedRandom,
}

Indicates how to choose between nodes of the search tree when no children have been evaluated.

Variants

Api

Consider the nodes in the order given by the search space API.

Random

Consider the nodes in a random order.

Bound

Consider the nodes with the lowest bound first.

WeightedRandom

Consider the nodes with a probability proportional to the distance between the cut and the bound.
