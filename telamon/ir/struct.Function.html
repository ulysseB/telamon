pub struct Function<L = LoweringMap> { /* fields omitted */ }

Describes a function and the set of its possible implementation.

The type parameter L indicates if the function is fozen (L = ir::LoweringMap) or not (L = ()). A frozen function cannot have any more IDs allocated. We use this to know the exact list of potential decisions before we run the exploration.
