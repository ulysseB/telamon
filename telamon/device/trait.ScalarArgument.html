Represents a value that can be used as a Function argument. Must ensures the type is a scalar and does not contains any reference.

Required Methods

fn t() -> Type

Returns the type of the argument.

fn raw_ptr(&self) -> *const c_void

Returns an operand holding the argument value as a constant.

fn gen_random<R: Rng>(_: &mut R) -> Self
